# Bura'a (برء) - Digital Health Records Management App 🏥

**Bura'a** is a comprehensive mobile application designed to digitize health records and simplify medical data management for individuals and families. This project was developed as a **Post-Graduation Self-Training Project** to demonstrate proficiency in modern mobile development using Flutter and Firebase.

## 🛠 What I Practiced & Mastered in this Project
In this project, I focused on applying industry-standard practices for mobile development:

* **Cross-Platform UI:** Built a responsive and intuitive Arabic user interface using **Flutter & Dart**.
* **Backend Integration:** Implemented real-time data handling using **Firebase Cloud Firestore**.
* **Authentication Flow:** Developed a secure login/signup system with **Firebase Auth**.
* **File Management:** Integrated **Firebase Storage** to allow users to upload and manage medical reports (Images/PDFs).
* **State Management:** Managed application states efficiently to ensure a smooth user experience.
* **Data Organization:** Designed a NoSQL database schema to handle complex family relationships and medical histories.
